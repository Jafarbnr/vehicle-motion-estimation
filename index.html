🚗 Vehicle Motion Estimation

This project demonstrates a portable 2D/3D Kalman Filter for vehicle state estimation, built in C with simulation, embedded support, CAN output, functional safety, and Python-based visualization.

📥 Try It

Download the repository or explore the interactive Jupyter Notebook demo.

📦 Features

  • 2D and 3D Kalman filter implementations in C
  • Real-time loop simulation
  • CAN-style output encoder
  • Functional safety (fault detection)
  • Embedded-ready version for STM32/RP2040
  • Python plotting scripts and notebook
